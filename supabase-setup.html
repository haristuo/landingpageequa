Equa Waitlist – Supabase Setup

Führe das folgende SQL in deinem Supabase-Projekt aus, um die Waitlist-Tabelle anzulegen.

→ Supabase SQL Editor öffnen 
-- Waitlist-Tabelle für Early-Access-Anmeldungen
CREATE TABLE IF NOT EXISTS public.waitlist (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  email text UNIQUE NOT NULL,
  created_at timestamptz DEFAULT now() NOT NULL,
  source text DEFAULT 'landingpage',
  converted_to_user boolean DEFAULT false
);

-- Row Level Security aktivieren
ALTER TABLE public.waitlist ENABLE ROW LEVEL SECURITY;

-- Policy: Anonyme Benutzer dürfen nur INSERT (hinzufügen), nicht lesen
CREATE POLICY "Allow anon insert" ON public.waitlist
  FOR INSERT
  TO anon
  WITH CHECK (true);

Schritte

  1. Klicke auf "→ Supabase SQL Editor öffnen"
  2. Klicke auf "SQL kopieren" und füge den Inhalt im SQL Editor ein
  3. Klicke auf Run (oder Strg/Cmd + Enter)
  4. Wenn "Success" erscheint, ist die Tabelle angelegt – die Waitlist funktioniert nun.
